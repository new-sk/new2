07. CSS가 뭐지?

HTML과 CSS를 분리하는 이유 : 가동성과 효율성 높이기 위해서

CSS 만들기

- 간단할 경우 : <style> </style> 태그로 꾸미기 할 영역 지정

- 대부분 별도 CSS 파일 만듦 : <link rel="stylesheet" href="style.css">


<style> p { color: red; } </style>

<p style="color:red">난 빨간색이야 </p>


구글링을 통해서 찾아서 하세요.

h1 {
font-size: 100px;
font-align: center;
} <!-- h1의 글자는 크기가 100px인 가운데정렬 -->


CSS 선택자

전체 선택자 (*)
* { color: red; } <!-- 모든 내용을 빨간색으로 만듦 -->

유형 선택자 (태그명칭) : 일반적으로 사용하는 것
h1 { color: red; } <!-- h1만 빨간 글자로 -->

아이디(#) vs 클래스(.) : 아이디는 고유한 값을 가짐 vs 하나의 클래스에 여러 태그 지정 가능, 하나의 태그에 여러 클래스 지정 가능
.my_class1 { color: red; }
<p class="my_class1 my_class2">여러 태그를 하나의 클래스로 묶을 수 있고, 태그 하나에 여러 클래스 적용 가능</p>

복합 선택자
조상자손선택자 (조상태그명칭 자손태그명칭)
div p { color: red } <!-- div 태그의 자손인 p 태그의 색깔을 빨간색으로 지정 -->
