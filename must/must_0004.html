04. 태그로 웹페이지 만들기 (1)

- 시작/종료 태그 : <태그명> 요소 </태그명>

- 시작 태그만 : <태그명 요소>


h, br/p/div/span, link, img, form, button, ol/ul

- h1 ~ h6 : <:h1></h1>, 6단계, 1은 하나만

- br : <br> : 문단 내 줄바꿈 : 시작태그만

- p : 문단 나누기 : <:p></p>

- div : 영역 나누기 : <:div></div>

- span : 줄바꾸지 않고 속성 바꿀 영역 지정 : <:span></span>

- oi, ui : 순서 유무에 따라 : <ol> / <ul>


- link : <link rel="속성" href="파일 경로"> CSS나 PNG 파일 등과 현재 HTML 문서와의 관계 정의

- a : <a href="https://www.google.com">visit google</a>

- img : <img src="경로" alt="설명" width="폭" height="높이"> 원본의 축소보다는 썸네일 활용후 필요시 고화질 사용 : alt 스크린리더가 화면 글 읽을 때 사용
