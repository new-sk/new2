09. CSS 레이아웃 : 플렉스박스

display 3번째 속성 : flex (더 다양한 레이아웃 만들고자 고안된 것으로 레이아웃을 유연하게, 자유자재로 배치 가능)
inline은 단순히 여백을 없애는데 사용 vs 축변경, 자식 간의 정렬 등 유연하게 박스 레이아웃 변경 처리
부모만이 플렉스할 수 있다.

부모인 body에 플렉스 적용

1. display : flex; <!-- 가로 길이에 맞추어서 div 배치 -->

flex-start, flex-end, center 등 3개는 공통으로 있는 속성값

2. (주축 정렬) justify-content : (5종류) space-between (등간격) space-around (가장자리 포함 등간격)

3. (교차축 정렬) align-items : (4종류) strech (긴 것에 맞추어 늘어뜨림)

body height 100%

--> 그렇지 않다면 요소 중에서 가장 큰 것을 기준으로 작업됨.

--> 화면 정중앙 위치하려고 하나, 요소중 제일 큰 것 기준으로 정렬되어 원하는 모양이 안 나올 수 있음
