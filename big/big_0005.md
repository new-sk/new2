# 05. 기초 통계

모집단(population) -> 표본(sample)
 - 표본추출(대표성을 가지도록)

1. 단순임의추출(simple random) : 난수 발생시켜,,,
2. 편의추출(convienence) : 조사의 편리하게 수행할 수 있는 사람 먼저
3. 할당추출(quote) : 모집단 특성 파악 후, 특성별 편의추출
4. 계통추출(systematic) : 모집단 리스트, 임의의 하나 선택 후 일정한 간격별로 규칙적으로 추출
5. 군집추출(cluster) : '군집'들로 나누어진 임의 표본 선택 (군집 나눈 후, 특정 군집 선택)
6. 층화추출(stratified) : 계층별 단순 임의 추출

<br>

측정
 - 명목척도 : 성별, 출생지
 - 순서척도 : 만족도, 선호도, 학년
 - 구간척도 : 온도, 지수
 - 비율척도 : 무게, 나이, 시간, 거리 (절대기준 0)

<br>

기술통계학(deescriptive statistics) : 통계집단의 여러 특성을 수량화하여,,, (평균, 분산,,,,)

추론통계학(inference statistics) : 수집된 자료 기반으로 모집단의 모수 추정 (가설검정)

<br>

데이터 수집 방법
 - 1차 데이터 : 분석자가 직접 수집 (면접, 설문)
   - 개방형 질문 (다양한 정보 획득 가능) <--> 폐쇄형 질문
   - 좋은 질문 설계
   - 응답편이 : 과대/과소, 응답자를 위한 답변
   - 무응답편이 : 무응답자들이 비슷한 특징을 가진다면,
 - 2차 데이터 : 신문, 리포트, 정부/기업의 출판물 등 이미 수집 공표된 데이터

<br>

빈도표(frequency distribution)
 - 양적자료 : 어떤 음료 먹었나?
 - 질적자료 : 물건 몇 개 샀나?
 - (절대)빈도(표), 상대빈도, 누적빈도, 누적상대빈도(cumulative relative frequency)
 - 그룹화된 빈도표
   . 계급의 수
   . 계급의 폭, 중간값, 경계(boundary : 두 계급의 가운데 값), 한계(limit : 상단, 하단)

<br>

그래프
 - 양적 : 막대, 파이
 - 질적 : 히스토그램, 산점도, 시계열그래프
 - 상대빈도 : 밀도함수

<br>

최빈값(mode) -> 질적자료 가능
평균(mean) -> 극단값 영향 받음  (산술평균, 결합평균, 가중평균)
중위수(median)

<br>

산포측정

범위(range) : 최대, 최소
사분위범위 : 최소, first, 중위수, third, 최대
 - IQR(interquatile range) : 3분위수 - 1분위수
 - five number summary
 - 상자그림(box plot), 상자수염그림, (수정) 상자그림

분산(variance)/표준편차(standard deviation)
변동계수 : 상대적인 변동성 측정(비교 위해) : 표준편차/평균

상관계수

관계는 산점도 -> 수치화 : 선형관계, 상관계수
(하나의 관측점이 다른 집단으로부터 멀리 떨어질수록 상관계수는 1에 가까워진다)
 - 단위의 영향없음 (미터, 피트로 변환해도 결과는 동일)
 - 극단값 영향 받음
 - 둘다 양적 변수에만 가능
 - 선형 관계만 설명해줌
