왜도(skewness) : 치우친 정도
 - 양수 : 오른쪽에 긴 꼬리 생성 (최빈값, 중앙값, 평균)
 - 음수 : 왼쪽에 긴 꼬리 생성 (평균, 중앙값, 최빈값)
 - 0 : 평균 = 중앙값 = 최빈값

<br>

첨도(kurtosis) : 뾰족한 정도
 - 양수 : 더 뾰족함

<br>

공분산(covariance)
 - 공분산을 표준화하면 상관계수
