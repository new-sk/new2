가설검정

  • 귀무가설(null, H0) vs 대립가설(alternative hypothesis, H1)

  • 유의확률

  • 기각역(critical region, 유의수준 이내인 부분) / 채택역(acceptance region)

비모수적 검정

  • 분포에 대한 가정 없이 진행,

왜도(skewness) : 치우친 정도

  • 양수 : 오른쪽에 긴 꼬리 생성 (최빈값, 중앙값, 평균)
  • 음수 : 왼쪽에 긴 꼬리 생성 (평균, 중앙값, 최빈값)
  • 0 : 평균 = 중앙값 = 최빈값

첨도(kurtosis) : 뾰족한 정도

  • 양수 : 더 뾰족함

공분산(covariance)

  • 공분산을 표준화하면 상관계수

확률
