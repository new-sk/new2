5. 선택자

5.1 선택자

<!DOCTYPE html>

<html>

<head>

<meta charset="UTF-8">

<title>CSS 선택자</title>

<style>

h2 { color:red; } <!-- type 태그명으로 사용 -->

.coding { color:green; } <!-- class 여러군데 사용 -->

#coding { color:yellow; } <!-- ID 한군데에서만 사용 -->

</style>

</head>

<body>

<!-- 우선순위 ID > Class > Type -->

<h2>TYPE : 기본 red</h2>

<h2 class="coding">CLASS(.) : green</h2>

<h2 id="coding">ID(#) : yellow</h2>

</body>

</html>


5.2 실제 사항

CSS 선택자

TYPE : 기본 red

CLASS(.) : green

ID(#) : yellow
