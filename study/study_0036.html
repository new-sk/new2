6. CSS 상속

6.1 CSS 상속

<!DOCTYPE html>

<html>

<head>

<meta charset="UTF-8">

<title>CSS 속성의 상속</title>

<style>

header { color: blue; }. <!-- 부모의 속성을 자식이 물려받는다. (일부) -->

header p { color: red; }. <!-- 특정 부모의 자식만의 속성 정의 방법 -->

p {color: green; }

</style>

</head>

<body>

<header>

<h2>header h2</h2>

<p>header p</p>

</header>

<footer>

<h2>footer h2</h2>

<p>footer p</p>

</footer>

</body>

</html>


6.2 실제 사항

CSS 속성의 상속

header h2

header p

footer h2

footer p
