3.3 저작권의 보호와 이용 (20.06.09, SW자산관리사)

#SW자산관리사

1. 보호의 내용

문화 및 관련 산업의 향상 발전에 이바지

저작권 (저작인격권+저작재산권) + 저작인접권

 

저작인격권 (3개)

  인간의 사상과 감정의 표현 -> 인격이 발현된 것 : 저작물에 대해서 저작인격권 부여

  일신전속적 권리 (타인에게 양도될 수 없슴, 사망이후 소멸, 명예훼손 인정 안됨)

  공표권 : 공표여부, 공표시기, 공표방법 등은 저작자에게 (공표하지 않을 권리도 있음)

  성명표시권 : 실명 또는 이명 표시, 저작물에 대한 창작자임을 표시하는 것

  동일성유지권 : 더 좋은 결과물로 변경되더라도 권리 침해 행위임, 단, 학교교육 목적상 부득이하다고 인정되는 범위 내에서는 인정

 

저작재산권 (7개, 사망후 70년 보호)

  복제권

  공연권 : 본인이 공연하거나, 타인에게 공연할 수 있도록 함

  공중송신권 : 방송, 전송, 디지털음성송신  (URL 링크는 침해 아님)

  전시권 : 미술/건축/사진,  공중에게 시각적으로 전시,  관람료 유무와는 무관

    공중에게 개방된 장소에서 항시 전시하는 것이 아니라면 복제물도 전시 가능 (복제물에도 권리 부여)

    위탁자의 동의 없이 초상화 등을 전시할 수 없음 (초상권 우선)

  배포권 : 권리소진이론(최초판매의원칙) : 일단 판매한 물건에 대해서 저작자의 배포권이 미치지 못함

    온라인에서는 공중송신권으로 다룸

    SW재판매 : 디지털권리소진은 검토의 여지가 있음

  대여권 : 음반이나 SW의 경우에는 영리목적의 대여는 불가능

  2차적저작물작성권 : 별도의 저작물이고 독자적으로 보호 받지만, 원저작자 권리 침해하면 안 됨

 

저작인접권

  실연자의 권리

    성명표시권, 동일성유지권

    복제권, 배포권, 대여권, 공연권, 방송권, 전송권, 보상청구권(방송사업자, 디지털음성송신사업자, 판매용음반)

  음반제작자의 권리

    뮤직비디오는 대상 아님 (음이 영상에 고정된 것은 대상이 아님)

    복제권, 배포권, 대여권, 전송권, 보상청구권

  방송사업자의 권리

    복제권, 동시중계방송권, 공연권 (직접적인 영리 목적 방송 시청 불가)

2. 보호기간 및 보호 방식

보호기간

   원칙 : 보상을 주어 계속 창작 유도 + 보호기간이 지나면 공공의 영역으로 편입 (2011년 50년->70년으로 개정)

   창작자를 알 수 없는 경우, 법인이 창작, 영상저작물 : 공표된 시점 기준 70년 보호,  데이터베이스는 5년 

   보호기간의 기산 : 다음해부터 기산

 

저작권의 등록

   무방식주의 채택 : 관청에 등록하지 않아도 권리가 부여됨

   저작권 등록할 수도 있음 : 권리가 부여된 것으로 추정 (반대 사실 입증되면 추정이 깨질 수도 있음)

   법정손해배상청구 : 저작물이 등록되어 있어야 손해배상 청구 가능

 

저작권의 위탁관리

   권리관계가 복잡해짐 : 누가 사용하는지 확인하기도 어려움 + 이용허락 받기도 어려움

   저작권위탁관리업 (저작권신탁관리업+저작권대리중개업)

   저작권신탁관리업 : 권리자로 구성된 단체, 영리목적 아님, 사용료 징수/분배 능력 보유, 문화체육부장관 허가 (2015년 기준 13개 단체)

 

외국인의 저작물 보호

  거주 외국인 + 맨 처음 대한민국 내에서 공표된 외국인,  상호주의

3. 저작권의 양도와 이용

저작권의 양도 : 여러 권리 중에 일부만 양도 가능,  2차적저작물작성권은 일반적으로 양도되지 않음 (SW는 예외, 유지보수가 따라다님)

 

저작물의 이용허락 : 이용할 수 있는 권리는 저작채산권자의 동의를 받지 않으면 제3자에게 양도할 수 없다

  특허법(실시), 상표법(사용), 저작권법(이용허락) : 라이선스

 

저작권 양도 및 이용허락에 관한 계약의 해석 : 계약시 명확하게 해야함

  새로운 이용방법이 생길 경우 (CD의 탄생, 이용허락범위도 명확하게 표시)

 

저작물 이용의 법정허락 : 국가에 보상금을 공탁하고 이용하는 것

  저작재산권자 불명인 저작물의 이용 : 상당한 노력을 한 경우에 가능

  공표된 저작물의 방송 : 공익상 필요한 경우, 저작재산권자가 허락을 거부하는 것이 정당하지 않다고 판단되는 경우

  판매용 음반의 제작 : 판매후 3년 경과 음반, 다른 판매용 음반으로 제작시
