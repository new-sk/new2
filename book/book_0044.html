#종교

할머니를 잃은 사라의 이야기

1. 그리스드교

 - 020 : 누구를 더 사랑하고 덜 사랑해서도 아니란다. 다만 그 아이는 잃어버렸다가 이제 다시 찾았잖니 (돌아온 탕아)

 - 029 : 하느림은 지붕 위에 사신단다. 네가 그 분을 찾으면 기쁘겠구나 (할머니의 마지막 편지)

 - 033 : 유태교(너희가 이 율법을 지킬 때에만 하느님이 너희를 사랑하신다)

         그리스드교 (하느님은 모두를 사랑하신다. 가장 중요한 것은, 너희가 하느님을 사랑하고 또한, 다른 모든 사람들을 사랑하는 것이다)

 - 036 : 예수님의 죽음과 부활 -> 죽음은 끝이 아니고, 천국에 들어갈 수 있다.

2. 유대교

 - 아프라함은 유목민 -> 가나안 정착 (오늘날의 이스라엘, 뜻 : 하느님이 우리를 위해 싸우신다)

 - 유곱 : 가나안 흉년 -> 이집트로 이주 -> 새 파라오가 노예로 부림 -> 어린 사내아이 죽임 -> 모세(물에서 건져내다)

 - 맏아들 죽일 것이다 -> 양의 피 문틀에 묻힘 -> 죽이자 항복함 -> 광야로 나감 -> 불만 가짐 -> 돌판 2개에 새긴 십계명

 - 만나고 헤어질 때 : 샬롬(평화)

 - 십계명 + 613개지 율법

3. 불교

 - 한 사람이 죽으면, 그에게서 작은 불꽃 하나가 다른 이에게로 넘어감 

 - 바라는 것이 하나도 없도록 노력

 - '니르바나'의 상태 (가장 행복한 상태)

 - 싯다르타(왕의 아들) -> 스스로 붓다라고 함 (깨달을 자)

 - 모든 고통을 견디면 알 수 있을 것이라 판단 -> 아니고, 보리수로 이동

 - 4성제, 8정도

 - 나는 실이 아니다. 나를 믿지 마라, 믿으려ㅁㄴ 다만 내가 하는 말을 믿어라 

 - 신의 존재에 따라 기도 <- -> 명상(불교)

4. 이슬람교

 - 무함마드 -> 천사들이 새로운 소식을 알려주면 꾸란에 기록

 - 다설개의 기둥(오주) : 무함마드가 사도, 하루 다섯번의 기도 + 라마단, 기부, 순례

 - 양쪽 어깨에 착한 천사와 못된 악마

5. 힌두교

 - 영혼 자체가 고스란히 환생한다 믿음 (불교와 유사하나,,,)

 - 카르마 : 선한 카르마를 많이 쌓으면 인간으로써 새로운 삶(환생)

 - 다르마 : 우리가 어떻게 살아야 할지 일러주는 규범

 - 묵샤(해탈) : 거대한 무의 상태

 - 신의 수 : 303 or 3003,,,

 - 최고의 신 : 브라흐만

 - 비슈누 : 9차례 세상을 구원하기 위해 나타남 (9번째 붓다, 향후 예수님의 모습으로 나타난다고 함)

 - 네팔 소녀신 쿠마리 : 1년에 딱 한 체례 궁 밖에 나오나 사람들의 경배 받기 위한 것이 (아무 말도 아 해야 경배자들에게 행운을 안겨준다고 함)
