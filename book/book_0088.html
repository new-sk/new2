뇌, 인간을 읽다 (19.11.06)

#뇌, #사고

 

00. 프롤로그 마음의 조각들을 모으며 : (내성주의) 마음을 연구하는 학문(마음에 접근할 권하조차 없음)  -> (행동주의) 행동은 관찰 가능, 측정하고 분석가능 (쥐, 비둘기 탐구) -> (행동 + 뇌과학)

 

01. 거만한 인간 : 머리가 부은 족속, 뇌의 절대적 크기는 코끼리, 고래가 4배 정도 큼,  상대적인 크기 사람 2.1% vs 쥐 3.2%, 작은새 8%,  대뇌화 지수,,,  신피질의 크기(언어, 사고, 기억) 새는 없음,  집단의 크기

 

02. 두 발로 걷는 유일한 영장류 : 허리/목/치질/탈장,출산의고통, 골반은 재형성(10+8개월),  손의 자유, 언어의 기원

03. 언어의 손짓 기원 : 언어가 손짓에서 진화했다는 설도 있음 -> 소리 : 호모사피엔스의 승리 : 손짓 언어를 말로 바꾸어서 (두번째 손의 자유)

 

04. 우리는 아프리카 인종이다 : 순수한 인류는 약 6만년 전의 대탈출에 동참하지 않은 아프리카인,  네안데르탈인 1~4% 동일, 데니소바인 20% (파푸어아뉴기니 멜라네시아인)

 

05. 편파적인 유인원 : 주의(attention)의 선택적 본성 (고릴라 실험), 주의는 내 마음대로 조절 가능,  전두엽 손상 : 적응 힘들어 함,  주의력 결핍장애 : 너무 쉽게 흐트러짐, 좌뇌는 오른쪽 주의, 우뇌는 왼쪽이나 양쪽 모두 주의 ,  편측무시

 

06. 왼손잡이에 대한 수수께끼 : 10%정도임,  아인슈타인, 벤저민 프랭클린, 파블로 피카소는 오른손잡이, 한쪽부모 20%, 양쪽부모 26%, 외손잡이 유전 vs 오른손잡이가 아닐 유전, 좌우대칭이나 비대칭적으로 진환 (한쪽 방향에서 포식자가 오면,,,)

 

07. 쪼개진 뇌, 쪼개진 마음 : 데카르트 심신이원론,  분리뇌 수술(난치성 간질환자 -> 좋아지긴 했지만), 좌우가 역할이 나뉜다는 논리는 갈수록 설득력이 떨어짐

 

08. 거꾸로 보면 알 수 없는 얼굴 : 반추상얼굴영역(측두엽에 있는 특별한 부위, 얼굴인식), 얼굴 뒤집으면 인식 못함, 음화로 표시해도 못 알아봄

 

09. 욕설을 하지 않겠습니다 : 아이들은 주위로부터 들어서 알게 됨, 전두엽이 욕하려는 충동 억누를 수 있느냐,  금기와 금기어가 없는 세상은 더 위험한 곳일지도,,,

 

10. 언어 본능 : 선교하러 갔다가 무신론자가 되다 (1977, 다니엘 에버렐, 브라질 오지 피다한족), 촘스키의 보편문법이 적용되지 않음, 내포문법,  언어는 말로 표현하기 위해 진화한 것이지 글로 표현하기 위해 진화한 것이 아니다 (문어)

 

11. 음악이 말을 한다 : 멜로디가 있는 반주는 감정을 전달하고 기억을 돕는 기능이 있다. / 절대음감  / 말이 통하지 않는 곳에서는 음악이 말을 한다

 

12. 틀린 기억 : 기억은 매우 선택적이다. / 설령 기억이 난다 해도 틀리기 일수다. / 우리의 뇌는 결코 사진기나 녹음기처럼 작동하지 않는다 / 일화기억(일상적인 일) vs 의미기억(한국의 수도는) / 기억상실증은 일화기억 훼손

 

13. 인간에게는 영원이 있다 : 인간만이 시간여행 가능(?) / 과거 회상, 미래 대비

 

14. 빨간 일요일 : 공감각 - 결합된 감각 (예: 글자와 색깔) / 쌍방향 아님 (글자보고 색깔 연상, 색깔본다고 글자 연상되지는 않는다)

 

15. 나는 네가 무슨 생각을 하는지 알아 : 마음이론(타인 생각 읽어냄-공감), 다른 동물들도 느낌(원숭이 줄 당기면 동료 아픔 -> 먹이 있어도 줄 안 당김) / 샐리-앤 검사(바구니와 상자에 구슬, 틀린 믿음),  협동과 기만의 미묘한 조합

 

16. 거울아, 뇌에 걸린 거울아 : 거울 뉴런(보는대로 따라 하는), 자폐증은 거울뉴런이 망가져서(추정)

 

17. 웃는다는 건 : 간지럼힘은 사회적으로 까다로운 거래임,  자기자신을 간지럽히는 행위,  상대에게 해를 입히지 않는다. 유머로 발전

 

18. 여왕의 옆모습은 어느 쪽일까? : 좌우구분 / 반구간반전(일어난대로 기록, 반대로 기록) - 반대편 공격도 대응하게 만듦

 

19. 뇌가 꼭 필요할까? : 뇌의 10% 만 사용 (근거는 없음), 아무리 작은 뇌손상이라도 거의 항상 정신적 기능에 영향을 미침, 전전두엽 손상 : 모든 부분 정상이나 책임감사라지고 불손해짐,  우리에게 한계를 부여하는 것은 뇌 공간이 아니라 문화다. (모든 언어 배울 가능성이 있지만, 어느 나라에서 태어났는지에 따라 배우는 언어 다름)

 

20. 전 거짓말을 못해요 : 거짓말과 허풍, 거짓말에 대해서 훨씬 더 흠잡기를 좋아한다.  제재가 없으면 거짓말쟁이가 항상 이득을 봄,  허풍의 보호 아래 거짓말을 숨길 수 있다는 뜻,  위약(플라시보)효과
