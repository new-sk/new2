2. 제우스의 사랑과 헤라의 질투

지혜의 신 메티스 -> 태어날 아들이 아버지보다 뛰어남 -> 매티스를 삼킴
머리가 깨질듯 아픔 -> 머리를 쪼개어 아테나 여신이 태어남 (어머니 없이 아버지에게서 태어난 존재) (전쟁의 신)
헤라도 혼자서 아이를 나음 -> 불구인 헤파이스토스 -> 하늘에서 땅으로 던짐 -> 바다의 여신 테티스가 구함
헤파이스토스가 의자를 헤라에가 선물함 -> 잠금장치 -> 뒤오니소스가 헤파이스토스를 데리고 옴 -> 화해
페니키아의 공주 에우로페 -> 소로 변해 접근 -> 크레테 섬으로 데려감 -> 자식 중 가장 유명한 이가 미노스 (훗날 죽은 자들을 심판하는 심판관이 됨)

처녀 신인 아르테미스 섬기는 요정인 칼리스토 -> 아이 가짐 -> 쫒겨나고 남자 아이 낳음 -> 헤라가 곰으로 만듦 -> 세월이 지나 아들이 곰을 보고 죽이려 함 -> 제우스가 아들도 곰으로 만들고 하늘의 별자리로 만듦 -> 새 별자리가 바닷물에 목욕하지 못하게 해 달라
독수리로 변해 데려옴 -> 술 따르는 남자 가뉘메데스 -> 여신 헤베가 헤라클레스와 결혼 후 빈자리였음
목성(주피터) - 위성 4개 (이오, 에우로페, 가뉘메데스, 칼리스토)

3. 젊은 신들의 성장과 사랑

아빠는 대부분 제우스, 그래서 엄마를 기억해야
아폴론 : 레토의 아들, 델로스섬에서 태어나 델포이로 옮겨감,그 곳에 살던 퓌톤(비단뱀) 물리침
에로스가 작은 활을 가지고 있는 것을 비웃음
에로스가 두 가지 화살을 쏨, 아폴론에게는 황금촉(사랑에 빠짐), 다프네에게는 납촉(사랑을싫어함)
아폴론에게 잡히자 강물의 신인 아버지에게 나무로 변하게 해달라고 함, 월계수

니오베 : 테바이를 다스렸던 암피온 왕의 아내, 자신이 레토보다 낫다고 자랑, 레토가 두 자식을 보내 죽임, 슬퍼 돌로 변함
인간 악타이온이 목욕하는 모습을 보자. 사슴으로 변하게 하고 자신의 사냥개에 죽임을 당함
헤파이스토스 : 다리가 불편함, 아내는 아름답기로 소문난 아프로디테 여신
아프로디테는 전쟁의 신 아레스와 바람을 피움, 헤파이스토스가 눈에 보이지 않는 그물로 현장에서 잡음
포도주의 신 디오뉘소스 : 어머니가 그를 낳기 전에 죽음, 헤라의 꼬임에 넘어가 죽음, 제우스가 허벅지에 아이를 심음, 그를 신으로 여기지 않은 사람들에게 가혹하게 대함 (뤼쿠르고스 왕, 테바이의 왕 펜테우스)

4. 페르세우스와 벨레로폰

다나는 아르고스의 공주, 아이를 낳으면 아버지 아크리시오스 왕을 죽인다는 신탁
청동으로 만든 탑에 딸 감금
비의 신이 제우스가 황금 비가 되어 그녀에게 내려 아이 생김, 아크리시오스는 엄마와 아기를 상자에 담아 바다에 보냄, 세리포스 섬에서 삶
세리포스 섬 왕 폴뤼텍테스가 다나에 차지하고자 함, 청년이 된 페르세우스가 방해됨
세 자매 괴물 고르곤의 머리를 가지고 오라고 함 (보는 사람을 돌로 만듦)
늙은채로 태어난 세 자매 괴물 그라이아이 찾아감, 눈 하나로 셋이 돌아가며 봄, 눈을 가로채서 요정 사는 곳 물어봄, 보복이 두려워 눈을 호수에 버림
요정에게서 3가지 선물 받음 (고르곤 머리를 담을 자루, 날개 신발, 보이지 않게 해 주는 모자)
헤르메스와 아테나의 안내를 받아 고르곤에게 감
둘은 영원히 죽지 않는 존재, 막내 메두사만 죽는 존재
메두사의 머리 잘라서 자루에 담고 모자 쓰고 달아남 -> 고르곤 추격하지만 못 찾음
메두사는 포세이돈 아이 임신 -> 날개 달린 말 페가소스, 크뤼사오르(활금의 칼) 작은 아이
돌아가는 길 : 아이티오피아에서 아름다운 소녀가 바다 괴물의 먹이되는 것 막고 결혼 - 카시오페이아 왕비의 딸 안드로메다 공주
이미 약혼한 피네우스의 부하들이 페르세우스 쫗아내려 함 -> 적들을 돌로 만듦, 안드로메다와 결혼, 폴뤼덱테스 일당도 돌로 만듦
요청들에게 빌린 물건 돌려주고, 메두사 머리는 아테나 여신에게 줌
아르고스로 돌아감, 외조부 아크리시오스는 도망침, 장례식 기념 원반던지기에 맞아 죽음, 이웃나라 티린스와 맞바꿔 다스림
북극성 가까이에 있는 별자리 이름 : 페르세우스, 안드로메다, 카시오페이아, 그녀의 남편 케페우스
벨레로폰 : 날개 달린 말 갖고 싶다고 신전에서 기도, 꿈 속에서 아테나 여신이 황금 재갈을 줌, 샘가에 있던 페가소스가 스스로 재갈을 받음
벨레로폰은 프로이토스 왕을 섬김, 욍비 스테네보이아가 반해 유혹하지만 넘어가지 않자 모함함
장인 이오바테스에게 보내서 죽이려 함
키마이라는 괴물 퇴치하라고 함 (머리 사자, 꼬리 뱀, 몸통은 염소, 염소머리에서 불 뿜음), 여러 어려운 일도 척척 해결, 자기 딸과 결혼시켜 나라를 물려줌
보디발 모티브 (창세기, 요셉이 보디발이라는 사람의 집에서 이야기)
벨레로폰의 소개장
벨레로폰은 너무 오만해져 페가소스를 타고 신들의 거처인 올림포스까지 감, 제우스가 말피리를 불어 페가소스에서 떨어져 다리 다침
이후 페가소스는 제우스를 변락 나르는 일을 함

5. 헤라클레스의 열두 가지 과업

제우스와 알크메네 사이에서 태어남
가족의 복수를 해 주면 함께 지냈겠다고 함. 남편 암피트뤼온은 테바이 왕에게 군사 빌려달라고 함 -> 엄청나게 무서운 여우를 퇴치해 주고 복수를 함
제우스가 암피트뤼온처럼 등장하여 아이 가짐 (헤라클레스)
불에 태워 죽이려 했으나, 제우스가 비를 내려 죄 없음을 알림 (이피클레스)
헤라는 대단한 아들 태어난 것에 분노하여 두고두고 괴롭힘
헤라가 얼른 페르세우스의 다른 자손 태어나게 함 (에우뤼스테우스) (아르고스 지역 전체를 다스리게 될 것이다)
뱀 두 마리를 요람에 보냄 -> 헤라클레스는 잡아 먹음 -> 제우스가 불멸을 존재로 만들고 싶어 헤라의 젖을 물게 함 -> 은하수
12가지 과제 (자신의 자식 죽인 죄값 -> 헤라가 그에게 광기를 불어 넣어서,,,)
01. 네마아의 사자 죽임 : 칼이나 화살로 뚫리지 않음, 목졸라 죽임, 발톱으로 가죽을 벗겨 몸에 두르고 다님
02. 머리가 여러 개 달리 물뱀 휘드라 퇴치 : 휘드라의 독을 화살촉에 바름, 1개 목을 베면 2개로 됨, 잘린 목을 불로 지짐
03. 황금 뿔을 가진 케뤼네이아의 사슴 잡아옴
04. 에뤼만토스 산의 멧돼지를 잡아옴
05. 스튐팔로스 호숫가의 무서운 새 떼를 퇴치 : 부리와 발톱이 청동, 독한 똥을 싸 농토 오염, 헤파이스토스가 만든 딸랑이로 새를 하늘로 날린 후 화살로 떨어뜨림
06. 아우게이아스 왕의 엄청나게 큰 외양간 청소 : 수십년간 청소 안 한 곳을 하루만에 청소, 인근의 강 두 줄기를 끌고와 간단하게 해결
07. 세상의 남쪽 끝 크레테로 가서 미쳐 날뛰는 엄청난 황소를 잡아옴
08. 세상의 북쪽 끝 트라키아로 가서 사람 잡아먹는 말을 끌고 옴, 마주 디오메데스는 사람을 잡아 말의 먹이로 주었다 함, 헤라클레스가 이 말 주인을 말의 먹이로 줌
09. 세상의 동쪽 끝 아마존 여왕의 허리띠를 구해 옴, 평화롭게 해결하던 중 헤라가 나타나 훼방을 부림
10. 세상의 서쪽 끝 게뤼온과 싸워 이기고 그의 소 떼를 몰고 옴
11. 세상 끝에 살고 있는 헤스페리데스에게 가서 황금 사과를 얻어 오는 것
12. 저승에 가서 머리 여럿 달린 개 케르베로스 데려 옴, 저승의 신 하데스가 무기로 쓰지 않는다면 데려가도 된다고 함
에우뤼스테스는 험악한 짐승을 잡아오면 무서워서 청동 항아리에 숨곤 함
활쏘기 시합에서 우승하여 공주 이올레와 결혼할 수 있었지만, 그쪽에서 말을 바꿈, 그 집안 아들 죽임
옴팔레라 여왕에게 노예로 팔려 일 년 동안 여자들이 하는 일 함
데이아네이라와 결혼 -> 다른 땅으로 떠나는 도중 켄타우로스 넷소스 만남 -> 거짓말을 함 (남편의 사랑을 잃게 되면 자기 피를 받아둔 옷을 남편에게 입히라고)
오이칼리아로 쳐 들어가 이올레 공주를 데리고 옴 -> 불앙함 느낀 데이아네이라가 피 묻은 옷을 줌 -> 고통속에 죽어감 -> 아버지 제우스에게서 받은 부분은 올림포스로 가서 헤라의 딸 헤베를 얻음

6. 테세우스와 아리아드네의 실

아테나이 왕 아이게우스는 델포이를 찾아가 신탁을 받음 -> "집에 가 닿기 전에는 포도주 자루를 풀지 말라"
트로이젠 왕 피테우스에게 찾아가 신탁 의미 물음 -> 피테우스는 자기 딸 아이트라와 결혼 시킴 -> 테세우스 태어남
아이게우스는 본향으로 돌아가며 바위 밑에 특별한 물건을 숨기며, 아들이 태어나 바위 들출 힘이 생기면 그 물건 가지라고 함 (샌들과 칼)
바다를 건너면 금방 아버지가 있는 아테나이로 갈 수 있었지만 헤라클레스처럼 영웅이 되고 싶어 육로로 이동

마라톤에 있는 황소를 잡아오라는 사명 (헤라클레스가 크레테에서 잡아온 것임) -> 황소를 무찌르고 돌아오자 독배를 준비함 -> 칼을 보고 깜짝 놀라 잔을 엎어 버림
아테나이는 크레테와 전쟁에 져서 매년 젊은이들을 미노타우로스의 먹이로 바침
미노타우로스 : 크레테의 왕 미노스의 아내 파시파에가 낳은 자식임
괴물을 낳자 명장 다이달로스를 시켜 꼬불꼬불한 골목으로 도저히 찾을 수 없는 미궁을 짓게 함
테세우스는 그 괴물을 퇴치하겠노라고 자진해서 크레테로 감 -> 미노스 왕의 딸 아리아드네가 첫 눈에 반함 -> 실 뭉치와 칼을 건넴 -> 괴물과 마주치자 칼로 죽이고 실을 따라가 미로에서 나옴 (어려운 일의 실마리 : 아리아드네의 실)
아리아드네와 도망침, 낙스스 섬에서 그녀를 버림, 신들이 헤어지게 함 -> 디오뉘소스 신의 아내가 됨
본의 아니게 아버지를 죽게 만듦 -> 깜빡하고 검은 돟을 단 채로 돌아옴(검은색/흰색) -> 아이게우스 바다에 몸을 던짐 (에게 해)
그가 참여했다는 모험이 너무 많아 "테세우스 없이는 안 된다"는 속담까지 있음
아마존 여자 하나를 납치하여 아내 삼음 -> 아들 태어남 (힙폴뤼토스)
미노스 왕의 딸 파이드라와 결혼 -> 파이드라가는 힙폴뤼토스와 사랑에 빠짐
힙폴뤼토스가 아프로디테를 무시하고 아르테미스만 섬겨 벌을 주려고 했다함
파이드라가 힙폴뤼토스에게 접근하려다 실패하자 모함하는 편자와 함께 자살함 -> 아들 쫓아냄
저승에 가서 하데스에게 붙잡힘 -> 망각의 의자 -> 케르베로스 잡으러 온 헤라클레스에게 구해짐
아테나이로 돌아오지만 다른 사람이 지배하고 있음 -> 스키로스 섬 망명 -> 리코메데스 왕이 절벽으로 떠밀어 죽게 됨
미궁을 지은 장인 다이달로스와 그의 아들 이카로스 -> 아테나이 사람인데 크레테에 삶 -> 미노스의 왕이 놓아주지 않자 날개를 만들어 탈출할 계획 세움
날개가 물에 눅눅해지지 않도록 너무 낮게 날지 말고, 밀랍이 태양의 열기에 녹을 수도 있으니 너무 높게 날지도 말라고 충고 -> 너무 높게 날다 떨어져 죽음 -> 젊은이들의 무모한 도전과 좌절

7. 아르고 호의 영웅들

보이오티아에 아타마스 왕 + 여신 네페레와 결혼하여 아이 둘 얻음, 부인 버리고 다른 여자와 결혼
새 아내는 전처의 두 아이 미워함, 흉년이 들자 자식을 제물로 바치려고 함 -> 네펠레는 하늘을 나는 양 보냄 -> 여자아이는 떨어져 죽고 아들 프릭소스는 콜키스에 도착
콜키스의 왕 아이에테스는 사위 삼음, 황금 양을 잡아 신께 제물로 바침, 그 가죽은 아이에테스에게 줌
한 세대 뒤 아이손 : 황금 양텰 가죽 얻기 위해 콜키스로 떠남
작은 아버지 펠리아스가 시킴 (아버지 아이손 몰아내고 왕이 된 사람) -> 샌들을 한쪽만 신은 사람이 나타나면 위험하다는 신탁 -> 아이손을 위험에 빠트리려고 함
할머니 모습을 한 헤라 여신을 업고 강을 건너다가 그렇게 됨
목수 아르고스가 아테네 여신의 도움으로 커다란 배 만듦 (아르고 호) -> 말하는 배
샘의 요청에게 납치된 필라스 구하려다 헤라클레스를 배를 놓침 -> 신의 뜻이라고 그냥 감 / 너무 무거워서 내리라고 했다는 얘기도 있음
눈 먼 예언자 피네우스 만나 하르피이아 좇아냄 -> 신의 뜻을 너무 많이 인간에게 알려줘 2가지 벌을 받음 -> 눈 멀고, 하르피이아라는 괴물(음식 빼앗거나 배설물 흩뿌림)
이상한 새 하르피이아를 쫓아 주고, 피네우스의 도움 받음
북풍 보레아스의 두 아들 칼라이스와 제테스의 가장 큰 공 -> 하늘을 나는 능력
피네우스의 보답 : 부딪히는 바위 통과하는 방법 가르쳐 줌 -> 먼저 비둘기를 날려서 무사히 통과하면,,,  -> 배 뒤부분 조금 부서짐 (아테나 여신이 손수 밀어주었다고 함)
옛사람들은 마법이 한번 통하지 않으면 그 힘을 잃는다고 생각했음
아이에테스 왕 만나 황금 양털 가죽 달라고 청함 -> 2가지 과제 내줌 : 불을 뿜는 첫동 소 두 마리에 멍에를 지워 밭을 갈고, 용 이빨을 땅에 뿌려 거기서 나오는 전사들과 싸우라고 함
아이에테스의 딸 메데이아 : 아이손에 매혹되어 직접 만든 신기한 약 -> 약 바르면 하루 동안 불길에도 안전, 힘까지 용솟음침
아이에테스 약속 안 지키고 기습 계획 세움 -> 메데이아가 황금 양털 가죽 훔쳐 함께 달아남 -> 메데이아는 자기 동생 압시르토스를 죽여 흘뿌림 -> 추격자들은 시신 수습하기 빠쁨
바다의 요정 세이렌 살고 있는 섬 지남 -> 오르페우스가 리라 연주해서 세이렌 노래 들리지 않게 함
탈로스 사건 : 청동으로 된 인간으로 하루에 세번씩 크레테 섬을 돌며 가까이 오면 바위 던져 공격 -> 발뒤꿈치 마개 : 메데이아는 약으로 추하게 해서 마개를 뽑아 죽임
펠리아스 : 자기가 죽을 때까지 기다리라고 함 -> 메데이아의 꾀 : 젊게 만들어 준다고 하여 죽임 (양)
왕을 너무 끔찍하게 죽여 코린토스로 추방됨 -> 아이손은 그 곳 공주와 결혼 -> 메데이아는 공주를 죽이고, 자기 아이들도 죽임 -> 용이 끄는 태양신의 수레를 타고 아테나이로 도망침 (테세우스 이야기와 연결)

8. 오이디푸스 가문과 아가멤논 가문

제우스가 소로 변해 업고 달아난 페니키아 공주 에우로페 -> 그의 오빠 카드모스는 동생을 찾아 나섰다가 지쳐 고향으로 돌아가는 것 포기하고 그리스 땅에 정착
도시 테바이 건설 (샘을 지키는 용과 싸워 이김, 아테네 여신이 나타나 용의 이빨 땅에 뿌리라고 함, 용사 태어나면 돌을 던짐, 그들끼리 싸우다 5명 남았을 때 싸움 멈추게 함)  (용의 후손이고 "씨를 뿌려 나온 사람들"이라고 자부함)
아레스 신의 자식인 용을 죽여 1년간 노예로 봉사 -> 아레스와 아프로디테 사이에 태어난 하르모니아(조화)와 결혼
암피온과 제토스는 테바이의 성벽 쌓음 :암피온이 리라 연주하면 돌들이 저절로 날아와 쌓임, 리라의 줄이 7개 -> 성벽의 문도 7개 -> 암피온 아내 니오베 -> 자식 자랑하다 모두 잃고 슬퍼 돌이된 여자
오이디푸스(카드모스의 후손으로 가장 유명) : 아버지 라이오스, 어머니 이오카스 -> 자기를 죽게할 것이라는 예언때문에 아이를 산에 갖다 버림
아이 버리는 일을 맡은 양치기가 이웃 코린토스의 양치기에게 아이를 넘김 -> 코린토스 왕의 양자 -> 오이디푸스 : 주워 온 아이라는 것을 알게 되자 진짜 부모님 알고 싶어 델포이에 신탁 구함
"너는 아버지를 죽이고 어머니와 결혼할 것이다"라는 신탁 받음 -> 겁이나 코린토스 반대방향으로 감 -> 좁은 길에서 다툼에서 노인 죽임(아버지)
테바이에서 스핑크스 수수께끼 품(스핑크스가 좌절하여 절벽에서 떨어져 죽음) -> 남편 잃은 테바이의 왕비와 결혼 -> 아이가 넷이나 태어남 -> 진실알게 되자 이오카스테는 자살, 오이디푸스는 장님이 되어 방랑 -> 딸 안티고네 시중
두 아들 에테오클레스와 폴뤼네이케스가 번갈아 가면 통치 -> 에테오클레스가 약속을 저버림 (동시에 죽음) -> 외삼촌 크레온 권력 잡음 -> 안티고네가 장례치룸 -> 동굴에 갇힘 -> 자살
탄탈로스 : 신들에게 자기 자식을 잡아 식사로 대접 -> 대지의 여신 데메테르는 조금 먹음(딸 페르세포네 잃은 수심으로,,,) -> 신들이 다시 몸을 만들어 주었지만 어깨는 상아로 채움 (펩롭스)
탄탈로스는 벌을 받음 : 가슴까지 차오르는 물에 몸을 담금 -> 물을 마시려면 내려감, 과일 먹으려는 살랑 달아남 -> 머리 위는 언제 떨어질지 모르는 바위
펠로폰네소스(펩롭스의 섬)에 정참 -> 힙포다메이라 공주와 결혼 (공주의 아버지 오이노마오스 : 먼저 말을 달려가게 하고, 제사 드리고 추격함, 잡히면 죽음)
마부 매수하여 이김 -> 매차의 축을 밀랍으로 만듦 -> 마차 부서져 오이노마오스 죽음 -> 마부 뮈르틸로스에게 약속한 보수 주지 않고 바다에 던져 죽임 -> 저주를 퍼부음
자식 세대에게 저주 본격적인 효력 : 미케네의 왕 아트레우스와 튀에스테스
튀에스테스는 아트레우스의 아내와 바람, 황금 양털 가죽 훔침 -> 아트레우스는 튀에스테스 잔치에 초대함, 그의 자식을 아비에게 먹임
다음 세대 : 튀에스테스의 아들 아이기스토스(그 땐 워낙 어려 살아남) vs 아트레우스의 자식 아가멤논과 메넬라오스 -> 트로이아 10년 전쟁 중 아이기스토스가 아가멤논 아내 클취타임네스트라와 바람 피움, 전쟁에서 돌아온 아가멤논 죽임
아가멤논의 아들 오레스테스와 누이 엘렉트라는 아이기스토와 어미니 클뤼타임네스트까지 죽임 -> 복수의 여신에게 쫓김 -> 아테나이에서 재판 -> 동수가 나와 풀려남

9. 트로이아 전쟁

바다의 여신 테티스를 두고 제우스와 포세이돈 겨룸 -> 아이를 낳으면 제우스의 능력을 능가하리라는 것을 앎 -> 인간과 결혼시킴 (펠레우스)
불화의 여신 에리스 여신이 초대받지 않은 결혼식에 참석하여 '가장 아름다운 이에게'라고 적힌 황금 사과를 던짐 -> 헤라, 아테나, 아프로디테가 서로 받으려고 함
제우스는 인간 파리스에게 떠 넘김 -> 헤르메스가 세 여신 데리고 파리스에게 감 -> 헤라는 강대한 권력, 아테나는 전쟁에서의 승리, 아프로디테는 가장 아름다운 여성 주기로 약속 -> 파리스는 아프로디테에게 사과 줌 -> 아가멤논의 동생이자 스파르타 왕인 메넬라오스와 결혼한 헬레네였음 -> 헬레네 트로이아로 데려와서 트로이아 전쟁 일어남
트로이아 전쟁 다룬 소설 : 호메로스가 지은 일리아스 -> 10년 전쟁 중 며칠 동안 일어난 일임
아킬레우스 전쟁 참여 거부 (아가멤논이 브리세이스라는 여인 빼앗아가서) -> 절친 파트로클로스가 죽음 -> 적장 헥토르에게 원수 갚음
헥토르가 죽고도 지원군으로 인하여 함락 안됨 -> 아마존 여전사 펜테실레이아는 아킬레우스에게 죽음 (아름다움에 슬퍼함) -> 에티오피아인 지원군 멤논(새벽의 신 에오스의 아들)
에오스와 테티스가 제우스에게 달려가 자기 아들 이기게 해 달라고 함 -> 헤르메스에게 시켜 두 사람의 운명을 저울로 담 -> 멤논이 먼전 죽고, 아킬레우스도 파리스의 화살에 맞아 죽음
아킬레우스의 무장 누가 차지할까로 다툼 : 물푸레나무 창, 불사의 말 두마리가 끄는 전차, 헤파이스토스가 만들어 준 방패와 갑옷 -> 오뒷세우스와 아이아스 -> 투표 결과 오뒷세우스 -> 아이아스는 자결
아킬레우스의 어린 아들 네옵톨레모스와 필룩테테스 데려옴 -> 필룩테테스는 포이아스 아들 -> 포이아스는 헤라클레스 죽을 때 장작더미 불지핀 이 (헤라클레스 활과 화살 받음) -> 필룩테테스 트로이아 가다가 중간에서 무인도에서 뱀에게 물림 -> 의사인 마카온 형제가 상처 치료해줌 -> 화살로 파리스 쓰러뜨림
아테나 여신 조각상이 지켜줌 -> 오뒷세우스와 디오메데스가 조각상 훔쳤지만 여전히 버팀 -> 목마작전

10. 오뒷세우스의 귀향

로토스 열매를 먹으면 계속 머물고 싶어진다고 함 -> 일행 중 몇 명 먹었으나, 억지로 끌고 옴
동료 열 두명과 함께 동굴 탐험 -> 새끼 양, 새끼 염소, 치즈, 우유 가득 -> 외눈박이 주인 거인 폴뤼페모스가 돌아옴 -> 엄청나게 큰 돌 굴려 동굴 입구 막음 -> 동료들 두 명씩 잡아먹음
오뒷세우스가 포도주 권함 -> 술을 마시고 기분이 좋아짐 -> 오뒷세우스는 자기 이름을 '아무도 아니'라고 가르쳐줌 -> 잠들자 나무창으로 눈을 찌름 -> 비명 소리에 놀라 동료 퀴클롭스 달려왔지만, '아무도 아니'야 라고 외침
바람의 왕 아이올로스 만남 -> 바람 자루 하나 줌(나쁜 바람 전부 가두어 둔 자루) -> 부하들이 자루 품 -> 다시 아이올리스 섬으로 휩쓸림, 도움 요청하지만 이번엔 거절
라이스트뤼고네스라는 거인들의 땅 -> 큰 바위 던져 배 부숨 -> 선원들을 식사꺼리로 가져감 -> 오뒷세우스는 밧줄을 끊고 간신히 탈출
키르케라는 요정 만남 -> 정탐 나간 부하 키르케 만나 식사 대접 받았지만, 돼지로 변함 -> 한 명 도망쳐 오뒷세우스에게 알림 -> 가는 도중 헤르메스 만나 몰리라는 약초 받음 -> 돼지가 되지 않고 키르케 위협하자, 애인이 되어달라고 함
애인이 되어주고, 돼지로 변한 동료 사람으로 돌려놓음 -> 1년이 지나자 떠나기로 함 -> 저승에 가서 예언자 테이레시아스 만나야 함
바다의 요정 세이렌 (오뒷세우스는 귀를 막는 대신 돛대에 몸을 뭆은 채 노래 들음) -> 무사히 통과하지 세이렌들은 바다로 떨어져 죽음
머리 여섯 개 달리 괴물 스퀼라와 엄청난 소용돌이 카륍디스 사이 지남 -> 간신히 태양신의 섬 트리나키아 닿음 -> 오뒷세우스가 잠시 자리 비운 사이 부하들은 신성한 소를 잡아 먹음 -> 바다로 다시 나간 이들은 갑자기 분 폭풍우에 오뒷세우스만 남기고 모두 죽음 -> 요정 칼륍소가 사는 섬에 닿음
칼립소에서 7년 동안 붙잡힘 -> 신들의 결정에 따라 헤르메스가 칼립소 찾아감 -> 뗏목 만들어 18일 동안 향해한 끝에 스케리아 당도 -> 빨래하러 나온 나오시키 공주 도움 받음 -> 왕은 딸과 결혼시켜주고 싶어하지만 선물까지 잔뜩 배에 실어 보내줌 -> 고향땅 이타케 섬으로 데려다 줌 -> 아테나 여신이 목동 모습으로 나타나 그를 늙윽 거지꼴로 바꾸어 돼지치기 에우마이오스 찾아가라 함 -> 그 곳에서 아들 텔레마코스 만남
오뒷세우스가 죽었다고 생각하고 그의 아내 페넬로페에게 결혼하자고 압박함 -> 오뒷세우스가 재산을 많이 모아 곧 돌아올 것이라고 전함 -> 늙은 유모가 발 씻겨주다 무릎 흉터 보고 오뒷세우스인 것을 알지만 입막음함 -> 활쏘기 시합으로 결혼하겠다고 함
