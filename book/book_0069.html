부재가 확 끌린다. 뻔하고 진 빠지는 심리게임 탈출 프로젝트

 

 

1부 우리는 왜 똑같은 걸로 싸울까

 

커뮤니케이션 분야에서 사용되는 심리게임 : 똑같은 패턴이 반복하여 작동함

016 : 심리게임 징표 : 원하는 바와 정반대의 결과, 소득없이 짜증만 남, 똑같은 얘기가 나오는데 진척된 느낌이 전혀 안듦

왜 게임이라고 할까? --> 근본적인 해결책을 찾는 것이 아니라, 자신의 존재감 나타내기 위한 방편으로 활용 (4장)

1. 왜 날 인정해 주지 않아?

에릭 번

사람은 인정자극이 필요하다. 긍정적이든 부정적이든. 무시당하는 것보다는 나으니깐. 언어적이든 비언어적이든

024 : 남에게 무시당한다고 느끼는 사람, 자극에 목마른 사람이 이러한 게임의 판을 벌인다

긍정적인 관계를 위해서는 : 나는 좋은 사람인가? 다른 사람들을 믿을 수 있는가?에 대해서 긍정적으로 답변할 수 있어야 함

2. 인정받고 싵은 세 사람

게임을 시작하게 되면 각자의 역할이 주어진다. : 피해자, 박해자, 구원자

피해자 : 난 아무것도 할 수 없어요. 책임을 전가한다.

박해자 : 욕구불만에 찌들어 화를 잘 냄

구원자 : 약자 보호, 하지만, 상대방을 어린애 취급하고 의존적인 인간으로 만듦

3. 그 사람은 왜 그럴까?

역할은 바뀐다.

어린시절의 경험 : 피해자의 위치는 아동

048 : 아이의 미숙함은 적어도 원칙적으로는 연령을 감안해 너그럽게 용인된다. --> 성인이 된 이후에도 책임을 타인에게 떠넘기로써 어린아이의 위치로 돌아감

050 : 부모가 부모 노릇을 감당하지 못하거나(피해자 역할 부모) 자기 역할을 넘어가 버리면(구원자 또는 박해자 역할 부모)

지나치게 까다로운 규율을 세워 놓고 아이가 여기서 어긋나면 엄하게 꾸짖는다. 

아이에게 부적절하게 도움을 주는 행동의 장기적 결과보다는 그런 행동으로 당장 얻게 될 좋은 부모 이미지에 더 신경 씀

모성적 돌봄과는 다르다. 만 7세 이후로는 그러한 태도는 독이 된다

책임회피, 욕구불만, 자기권능의 확인

4. 지금 이게 내 탓이란 말이야?

이 게임에서는 “책임”이라는 공이 움직인다

피해자가 능숙하게 막아서 책임을 박해자, 구원자에게 보냄 --> 책임을 죄의식으로 둔갑됨 --> 게임의 본질상 실직적인 해결책을 찾기보다는 죄의식을 조장하는데 초점이 맞추어짐

죄의식의 해독제는 공정하고 객관적인 책임 분배다. --> 공을 패스하지 말고 조각내라 (게임에 참여하지 말라) --> 내 몫의 책임에 한하여, 나에게 그 책임을 감당할 힘이 있을 때 알맞은 행동을 취하라 (에너지의 중심을 잘 잡는 것이 대단한 휴식이다)

5. 나를 건드리는 그 한마디

서로의 말에 귀 기울이고 서로 존중하면서 싸울 수도 있다

심리게임을 촉발하는 행동들,,,

전체화 : 과장하기, 빈말로 때우기, 밑도 끝도 없이 갖다 붙이기

평하 : 남들과 비교해서 깎아내리기, 딱지 붙이기, 부정, 조롱, 심문조의 이유 추궁(꾸지람 당하는 아이), 불변의 진리로 밀어붙이기, 위협

지레짐작

책임전이

6. 어디 한번 해 보자는 거야?

1단계 : 떡밥 던지기 : 게임에 잡아들이는데 사용: 너 숙제 다했니?

2단계 : 약점 찌르기 : 잠재적 게임 상대의 약점을 놓치지 않고 공략하여 자동 반응을 불러일으킴

3단계 : 자동반응 : 균형감각을 잃고 자기변호에 급급해짐

4단계 : 역할정하기

5단계 : 덪이 설치된 교류 : 겉으로 보기에 정중하고 아무 문제 없더라도 엄청난 공격성 내포 가능

6단계 : 극적반전 : 꼼짝못하게 하는 한 마디 --> 참가자 중 한 사람은 처지가 바뀌면서 게임이 끝난다

7단계 : 혼란과 부정적 효과 --> 게임이 끝나면 패자와 패자만 남는다

8. 혹시 내가 문제는 아닐까?

심리게임 단계 : 대수롭지 않은 단계, 자주 부딪치는 사이, 고도의 덪이 설치된 교류 등 감당 안되는 상태

144 : 자기가 하는 심리 게임을 파악하고 해체하는 것 자체가 자기 자신과 충분히 거리를 두고 스스로에게 정직해져야만 가능 (자극적 관계 : 사람하는 사람이 나를 아프고 힘들게 하는 게 당연하다고 배웠을 수 있다) 

9. 습관적 다툼을 끝내는 심리 처방

오해는 하나씩 푼다

갈등이 불거졌을 때는 6시간 후 행동에 나선다

당사자와 직접 말하라

사실에만 입각하라

적당한 선과 예의를 지켜라

나를 보살피고 나의 욕구에 귀를 기울이자

표현되지 않는 요구는 만족시킬 필요가 없다

10 어떻게 역할에서 벗어날 것인가?

피해자라면 : 내가 나를 다잡고 노력해야만 나답게 살 수 있다.

박해자라면 : 나의 욕구 불만의 근본 원인을 찾는다. --> 나의 상처와 욕구를 보살피자

구원자라면 : 나의 에고를 충족시킬 수 있는 다른 건설적 방법 찾기

상대가 피해자라면 : 그 사람이 구체적으로 바라는 바를 분명히 말로 표현하게 하고 자기 마음속에서 답을 찾게 하라

상대가 박해자라면 : 내가 정말로 원하지 않는 상처는 내 것이 될 수 없음을 명심하자. 차분하게 내가 할 말을 한다????? 강압적이지는 않되 단호하게 나간다

상대가 구원자라면 : 도움을 과감하게 거절하라. 고마움은 표현하라

11. 도와주고도 육먹는 당신이라면

정확히 원하는게 뭐죠?

도움에는 늘 마감기한이 있어야 한다

정당한 보상이 필요하다. (눌러앉은 여동생에게 생활비 내라고 한다든지,,,)

도움은 절반만 : 도움 받는 사람도 제 몫을 하고 적극적으로 자기 구제에 협력하는지 짚어야 함

자, 이제 물고기를 잡아 보라고 : 사람은 원래 편하게 살 수 있는 방향으로 적응(피해자가 당당하게 물고기를 요구함)

12. 난 이제 네가 힘들지 않아

음, 하나도 제대로 해결할 방법을 모르겠는데,,,

 - 특정 역할들에서 벗어나는 법을 안다

 - 공격성을 자극하지 않으려면 어떤 식으로 행동해야 하는지 알게 되었다

 - 건강한 도움의 손길을 내밀 수 있다

 

심리게임에 도가 튼 사람들을 상대한다면 구원은 하나뿐이다. 도망가라
