육아는 과학이다 (10.01.00)

- 정서적으로 안정된 아이를 키우기 위해서는 내가 좀 더 안정되어야…

1. 아이의 성장과 발달이 시작되는 뇌

- 상위뇌  하위뇌 (분노.두려움.분리불안.탐색.보호.놀이…)

- 생후 몇 년 동안 하위뇌가 지배  원시적인 충동

- 부모가 일관되게 자상한 반응  아이 상위뇌 발달에 도움

- 아이의 격한 감정 충분히 달래주어야 함

- 살아가면서 겪게 되는 불가피한 고통으로부터 구해줄 수는 없지만, 효과적인 스트레스 조절체계와 항불안 화학체계가 아이의 뇌에 확고하게 자리잡게해서 삶의 질을 높일 수 있다

- 부모가 자신의 격한 감정을 다스린 후에. 아이의 격한 감정 다스림

2. 아이의 생존욕구, 울음과 분리불안

- 생후 첫 3개월 동안 가장 많이 운다

- 계속 울게 내버려 두면  지쳐 잠든다  스트레스 반응체계 손상

- 달래는 방법 : 신체접촉/마사지, 손가락빨기, 체온 따뜻하게(목욕포함), 규칙적으로 흔들기, 낮은진동음(출생직후), 호기심자극, 지나친자극삼가

- 달래기 어려운 아기 : 엄마가 먼저 위안을 받아야할지도…

- 분리불안 : 6~8개월 ~ 5세 이후까지 지속되기도 함

- 스트레스 예방 접종 : 알맞은 정도의 스트레스 상황일때만 의미있음

- 아이와 대화하기를 좋아하는 사람 선택하기

- 아이를 떨어뜨릴 때는 익숙한 사람에게

- 생후 1년이 지나면, 울때마다 보살핌을 받은 아이가 덜 운다

3. 아이의 뇌가 편안하고 건강한 수면훈련

- 수면주기 50분  자주 깬다 (어른은 90분)

- 5세 이하 아이 25% 수면 문제 가지고 있음

- 생후 3개월 20%에서 심하게 보챔

- 아이의 두려움.불안을 진지하게 받아들여야 함

- 엄마와 함께 잔 아이 SIDS 감소 (영아돌연사)

- 억지로 따로 재우지 않는다

- 애착이론 : 애착 - 절망 - 탈애착

4. 아이의 삶을 풍요롭게하는 호르몬

- 오피오이드.옥시토신  코르티솔

- 다정한 신체접촉은 어떤 방식이든 긍정적 효과 지님

- 기쁨 (아드레날린. 도파민/오피오이드)

- 아이는 눈피함 : 피곤하거나 대화 멈추고 싶을 때

- 의욕 (도파민)

- TV앞이나 집에서 뒹굴 때 탐색체계 비활성화

- 새로운 경험을 선물한다

- 자유놀이와 탐구학습

- 놀이체계 활성화

- 과잉행동장애(ADHD)(5~7%)  약으로 놀이를 대신할 수 없다

5. 말썽과 떼쓰기, 아이의 뇌에 답이 있다

- 원인 : 피곤.허기.음식(첨가물)

- 충분한 자극을 받지 못하면 아이 스스로 사건 창출

- 떼쓰기는 아이 뇌를 조형하는 중요한 순간임

- 떼쓰기는 힘겨루기가 아님  아이의 입장에서 생각 

- 2유형 : 스트레스에 의한 것(달래주어야 함), 황제(무시,설명설득불필요)

- 운동 충동 발산할 수 있도록 도와주어야 함

- 정리정돈 놀이

- 감정은 이해하되 “안돼”

6. 아이의 건강한 정서를 키우는 훈육법

- 체벌 : 폭력을 가르친다

- 고함/잔소리 : 복종.지배를 배움

- 고함칠 때는 따로 있다

- 경계를 정해주지 않으면 한계가 어디인지 모른다

- 경계를 넘을 때 : 단호하게 주의. 그 결과 제시

- 잘한 행동에 대해서는 칭찬

- 관심 끌기 위한 행동은 무시

- 5세 이하 아이 : 설득이 아니라 간단하게 지시 내림

- 선택과 결과 제시

- 타임인 : 상황 정리된 후 왜 그런 행동했는지 파악. 숨은 감정 찾기

- 생각하는 의자

- 타임아웃 : 한 살에 1분. 최후의 수단으로 남용금물

- 감싸안기

7. 행복한 아이로 자라게하는 사랑의 힘

- 한결같고 변함 없어야 함

- 애착을 느끼는 사람은 소수에 불과

- 더 많이 사랑하고 표현

- 아이가 놀이를 주도하게 한다

- 부모도 화 낸다  다시 연결하는 것 중요

8. 따뜻한 인간관계를 맺는 능력, 사회지능

- 다른 사람과 관계를 맺는 능력

- 타협하고 해결하고 훌륭한 구성원이 되는 능력

- 동정과 관심을 갖고 표현하는 능력

- 아이가 때를 쓸 때 차분하게 대응

- 자폐증 초기에 치료하면 75% 성공

- 격한 감정을 떼쓰기가 아닌 말로 표현하게 됨

- 억울한 일을 당했을 때 자연스럽게 도움 요청

- 성격 유전자는 경험에 의해 변할 수 있음

9. 행복한 육아를 위한 부모 자신 돌보기

- 엄마 : 충분한 휴식 필요  그렇지 않을 경우 문제 야기

- 휴식이 충분하지 않으면 : 상위뇌 최선의 방책아닌 하위뇌의 분노처리

- 식품으로 보충 (비타민 B6 등)

- 재충전 : 아이에서 벗어나 자신만의 편안하고 즐거운 시간 보냄 (혼자 또는 마음맞는 누군가와)

- 아이와 있어도 충분히 행복하지만. 정서적 안정은 감정이입되는 어른과

- 심리치료사 : 배우자 등은 상호적 관계라는 단점이 존재함
