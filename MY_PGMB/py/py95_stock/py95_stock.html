class 명 : my95_stock

파일의 위치 : 원칙적으로 파이썬 소스코드가 있는 곳에 기본적으로 위치하고, 상대 경로임

  • 환경 변수에서 정의되는 DB 파일과 엑셀 파일이 존재하고
  • 환경 설정 파일 : my95_stock.ini
  • 실행 파일 : my95_stock.py (클래스 함수를 제외한 메인 영역)

환경설정 파일 변수 :

  • 테스트모드 여부 : 테스트모드(디폴트값) 또는 실행모드
  • DB 명 : 디폴트는 my_stock_2.db
  • 엑셀파일 명 : 디폴트는 my_stock_excel.xlsx
  • 파일 위치 : 디폴트는 D:\my_stock\

생성자 함수

  • OS가 무엇인지 확인
  • 맥이면 무조건 테스트 모드
  • 파일의 위치는 테스트 모드일 경우에는 파이썬 소스 코드가 있는 곳
  • open_db 함수 호출

디비 오픈 및 기본 SQL문 수행 함수 : open_db

  • 디비를 오픈 : 에러가 발생하면 종료
  • run_select 실행해서 결과값을 저장
    • 예제
    • sql문 저장하는 변수 : sql_dfi
    • 함수 호출 후 결과 값 저장 변수 : self.dfi

sql문을 실행하고 결과값을 데이터프레임으로 만들어 리턴해 줌 : run_select

  • 입력값은 sql문이야
  • 입력값 중에 하나는 결과값이 empty 허용 여부야
  • sql문이 오류가 발생하면 종료해줘
  • empty를 허용하지 않는데, 결과값이 empty이면 종료해줘
